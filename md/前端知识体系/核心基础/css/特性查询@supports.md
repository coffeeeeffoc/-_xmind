# 特性查询@supports

## 指定依赖于浏览器中的一个或多个特定的CSS功能的支持

## 语法

### 由一组样式声明和一条支持条件构成

### 支持条件由一条或多条使用 逻辑与（and）、逻辑或（or）、逻辑非（not）结合的名称-值对（name-value pair）组成

### 使用圆括号调整操作符的优先级

## 声明语法

### CSS 声明，也就是一个 CSS 属性后跟一个值，中间用冒号分开

### 例子

- @supports (transform-origin: 5% 5%) {}

## 函数语法

### 支持函数

### 目前只有selector

- 是否支持经过测试的选择器语法
- 例子

	- @supports selector(A > B) {}

## 逻辑操作符

### and

### not

### or
